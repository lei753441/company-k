# IT開発会社 管理システム - 業務モジュール一覧

## ユーザーロール

| ロール | 説明 |
|--------|------|
| 管理者 | システム全体の管理、ユーザー・権限管理 |
| 営業 | 案件管理、顧客対応、マッチング |
| 事務 | 勤怠集計、請求・支払管理、契約管理 |
| 社員 | 勤怠打刻・報告、キャリア管理 |
| 協力会社/個人事業主 | 案件情報閲覧、勤怠報告、請求書提出 |

---

## 業務モジュール一覧

### 1. 認証・権限管理
- **主なユーザー**: 全員
- **機能概要**: ロールベースアクセス制御（RBAC）、ログイン認証、SSO対応
- **主要機能**:
  - ユーザー登録・編集・無効化
  - ロール定義・権限設定
  - ログイン履歴・セキュリティ管理

### 2. 社員管理
- **主なユーザー**: 管理者・事務
- **機能概要**: 社員の基本情報・雇用情報・所属管理
- **主要機能**:
  - 基本情報（氏名、連絡先、住所）
  - 雇用形態・入社日・所属部署
  - 契約条件（給与、稼働条件）

### 3. スキル・キャリア管理
- **主なユーザー**: 社員・管理者
- **機能概要**: エンジニアのスキルシート・資格・経歴・稼働状況管理
- **主要機能**:
  - スキルシート作成・更新（言語、FW、経験年数）
  - 資格・認定情報
  - プロジェクト経歴
  - 稼働状況（空き・稼働中・稼働終了予定日）

### 4. 顧客・取引先管理
- **主なユーザー**: 営業・管理者
- **機能概要**: 顧客企業・担当者・取引履歴の管理
- **主要機能**:
  - 顧客企業情報（会社名、住所、業種）
  - 担当者情報
  - 取引履歴・商談履歴

### 5. 協力会社・個人事業主管理
- **主なユーザー**: 管理者・事務
- **機能概要**: 外部パートナーの登録・契約条件・評価管理
- **主要機能**:
  - 協力会社・個人情報登録
  - 契約単価・支払条件
  - 評価・実績履歴

### 6. 案件管理
- **主なユーザー**: 営業・管理者
- **機能概要**: 商談から受注・終了までの案件ライフサイクル管理
- **主要機能**:
  - 案件登録（顧客、要件、期間、単価）
  - ステータス管理（商談中・提案中・受注・進行中・完了・失注）
  - 案件DB検索・フィルタリング

### 7. マッチング
- **主なユーザー**: 営業
- **機能概要**: 案件と社員・協力会社のマッチング・提案管理
- **主要機能**:
  - スキル・稼働状況によるエンジニア検索
  - 提案リスト作成・管理
  - マッチング結果の記録・フィードバック

### 8. 契約管理
- **主なユーザー**: 事務・管理者
- **機能概要**: 顧客契約・業務委託契約のライフサイクル管理
- **主要機能**:
  - 契約書作成・管理（顧客向け・協力会社向け）
  - 契約期間・更新アラート
  - 電子署名連携（将来対応）

### 9. 勤怠管理
- **主なユーザー**: 社員・協力会社・事務
- **機能概要**: 出退勤記録・勤務表提出・承認フロー・工数管理
- **主要機能**:
  - 出退勤打刻（社員向け）
  - 月次勤務表作成・提出（社員・協力会社）
  - 勤務表承認（事務・管理者）
  - 残業・休暇管理

### 10. 請求・支払管理
- **主なユーザー**: 事務・管理者
- **機能概要**: 顧客への請求書発行と協力会社への支払管理
- **主要機能**:
  - 請求書作成・発行（顧客向け）
  - 支払管理（協力会社・個人事業主向け）
  - 入金確認・消込
  - 月次締め処理

### 11. 書類管理
- **主なユーザー**: 全員
- **機能概要**: 契約書・請求書・スキルシート等のドキュメント一元管理
- **主要機能**:
  - ファイルアップロード・ダウンロード
  - カテゴリ・タグ管理
  - 権限別アクセス制御
  - バージョン管理

### 12. 通知・ワークフロー
- **主なユーザー**: 全員
- **機能概要**: 承認フロー・期限アラート・メール通知の一元管理
- **主要機能**:
  - 承認ワークフロー（勤務表・経費・契約更新）
  - 期限アラート（契約更新、請求締め日等）
  - メール・アプリ内通知

### 13. ダッシュボード・レポート
- **主なユーザー**: 管理者・営業・事務
- **機能概要**: 経営KPI・稼働率・売上・案件進捗の可視化
- **主要機能**:
  - 稼働率レポート（社員・協力会社）
  - 売上・利益レポート
  - 案件進捗サマリー
  - カスタムレポート出力（CSV/PDF）

### 14. 経費管理
- **主なユーザー**: 社員・事務
- **機能概要**: 交通費・経費申請と承認フロー
- **主要機能**:
  - 経費申請・領収書添付
  - 承認フロー
  - 月次精算・集計

---

## モジュール間の主な連携

```
顧客管理 ──→ 案件管理 ──→ マッチング ──→ 社員/協力会社管理
                 ↓                              ↓
             契約管理                       勤怠管理
                 ↓                              ↓
             請求管理 ←────────────────── 請求・支払管理
```
