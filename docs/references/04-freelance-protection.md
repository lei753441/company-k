# フリーランス保護新法

## 概要

| 項目 | 内容 |
|------|------|
| 正式名称 | 特定受託事業者に係る取引の適正化等に関する法律 |
| 通称 | フリーランス・事業者間取引適正化等法 / フリーランス保護新法 |
| 施行日 | 2024年11月1日 |
| 所管 | 公正取引委員会・厚生労働省・中小企業庁 |

---

## 対象者の定義

### 特定受託事業者（フリーランス）
業務委託を受ける事業者のうち、**従業員を使用しない者**（個人または法人）。

### 特定業務委託事業者（発注者）
**従業員を使用する**事業者がフリーランスへ業務委託を行う場合に本法の義務が生じる。

> 本システムでいう「個人事業主（フリーランスエンジニア）」への発注が主な対象。
> 協力会社（法人で従業員を雇用）への発注は下請法が適用される場合もある。

---

## 発注事業者の主な義務

### 1. 書面（電磁的方法）による取引条件の明示義務（第3条）

業務委託をする際、**直ちに**以下の事項を書面または電磁的方法で明示する義務：

| 必須明示事項 | 内容 |
|------------|------|
| 業務内容 | 委託する業務の具体的内容 |
| 報酬額 | 金額または計算方法 |
| 支払期日 | 報酬の支払期日 |
| 発注事業者の名称等 | 会社名・担当者 |
| 業務委託をした日 | 契約締結日 |
| 成果物の内容 | 納品物・納期（ある場合） |

### 本システムへの適用（08-contract.md）
- 個人事業主との契約書に上記全項目を必須記載
- 電子契約で交付・保存

---

### 2. 報酬の支払期日（第4条）

| 区分 | 期限 |
|------|------|
| 支払期日の設定 | 業務委託日から起算して**60日以内**（かつできる限り短く） |
| 具体的な支払期日 | 当月末締め翌月末払いなど、明確に定めること |

### 本システムへの適用（10-billing.md）
- 個人事業主への支払サイクルは最長60日以内に設定
- 支払期日を超過した場合にアラート表示

---

### 3. 禁止行為（第5条）

継続的業務委託（6ヶ月以上）の場合、以下が禁止：

| 禁止行為 | 内容 |
|---------|------|
| 受領拒否 | 正当な理由なく成果物の受領を拒否すること |
| 報酬の減額 | 一方的な報酬の減額 |
| 返品 | 正当な理由なく成果物を返品すること |
| 買いたたき | 著しく低い報酬での委託 |
| 購入強制 | 不当な購入・役務利用の強制 |
| 不当な経済上の利益提供強制 | 金銭・労務等の提供強制 |
| 不当な給付内容の変更・やり直し | 一方的な業務内容変更・やり直し強制 |

### 本システムへの適用（06-project.md / 08-contract.md）
- 単価変更・業務内容変更時は相手方の同意記録を必須化
- 変更履歴・同意日時を監査ログとして保存

---

### 4. 育児介護等への配慮義務（第13条）

継続的業務委託（6ヶ月以上）の場合、育児・介護・病気治療等と業務の両立に関する申出があった場合に配慮する義務。

---

## 偽装請負リスクへの対応

本法とは別に、**労働者派遣法**上の偽装請負（指揮命令関係のある業務委託）は違法。

### 本システムへの適用（05-partner.md / 08-contract.md）
- 業務委託形態が「成果物指示」か「工数（時間）指示」かを記録
- 指揮命令関係が疑われる契約形態の場合、警告を表示
- 契約書に指揮命令関係が生じない旨の条項を明記
