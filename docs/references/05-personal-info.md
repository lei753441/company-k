# 個人情報保護法・マイナンバー法

---

## 個人情報保護法

### 概要

| 項目 | 内容 |
|------|------|
| 正式名称 | 個人情報の保護に関する法律 |
| 所管 | 個人情報保護委員会 |
| 最終改正 | 2022年4月施行（3年ごと見直し） |

### 個人情報の定義

生存する個人に関する情報で、氏名・生年月日その他の記述等により特定の個人を識別できるもの。

### 要配慮個人情報（センシティブ情報）
取得に本人同意が必要な情報：
- 人種・民族・社会的身分・病歴・犯罪歴・障害の有無 等

---

### 本システムで取り扱う個人情報

| 情報種別 | 該当データ | 配慮レベル |
|---------|-----------|----------|
| 基本情報 | 氏名・住所・生年月日・連絡先 | 通常 |
| 雇用情報 | 給与・雇用形態・契約条件 | 高 |
| 金融情報 | 銀行口座番号 | 高 |
| 識別番号 | マイナンバー | 最高（番号法適用） |
| 経歴情報 | スキル・資格・職歴 | 通常 |
| 生体・位置情報 | 打刻時刻・勤務場所 | 通常 |

---

### 本システムへの適用要件

#### 社員管理（02-employee.md）
- 給与・銀行口座は暗号化保存（AES-256）
- 退職後の個人情報保持期間：7年（法定保存期間に準拠）
- 個人情報閲覧権限は管理者・事務に限定

#### 書類管理（11-documents.md）
- 個人情報含有書類（CAT-001〜004）は権限保有者のみアクセス可
- 署名付きURL（有効期限15分）による直接アクセス防止
- ダウンロードログの記録・保存

#### マッチング（07-matching.md）
- 候補者の個人情報（住所・生年月日等）はマッチング画面に表示しない
- 候補者への通知に単価情報を含めない

#### ダッシュボード（13-dashboard.md）
- カスタムレポートで個人情報を含む列（住所・給与等）は管理者のみ選択可

## マイナンバー法（番号法）

### 概要

| 項目 | 内容 |
|------|------|
| 正式名称 | 行政手続における特定の個人を識別するための番号の利用等に関する法律 |
| 所管 | デジタル庁・個人情報保護委員会 |
| 施行日 | 2016年1月 |

### 利用目的の限定

マイナンバーは**法定された目的のみ**で使用可能：
- 税務関係書類（源泉徴収票・支払調書等）
- 社会保険・雇用保険手続き

マッチングや一般業務管理目的での利用は不可。

---

### 本システムへの適用要件

#### 社員管理（02-employee.md）
- マイナンバーは**暗号化保存（AES-256）**
- アクセスはアクセスログを**必ず記録**（法的要件）
- 閲覧権限は最小限（事務・管理者の一部のみ）
- 利用目的：源泉徴収票・社会保険手続きのみ
- 退職後の保管期間：源泉徴収票等の法定保存期間（7年）終了まで

#### 協力会社・個人事業主管理（05-partner.md）
- 個人事業主への報酬支払時、源泉徴収のためマイナンバーを取得する場合がある
- 同様に暗号化保存・アクセスログ必須
