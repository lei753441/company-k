# 労働基準法・36協定

## 概要

| 法令 | 所管 | 主な適用対象 |
|------|------|------------|
| 労働基準法 | 厚生労働省 | 雇用契約を結ぶ全社員 |
| 労働安全衛生法 | 厚生労働省 | 全社員（健康管理） |
| 36協定（時間外・休日労働に関する協定届） | 労働基準監督署 | 時間外労働を行わせる場合 |

> ※ 協力会社・個人事業主（業務委託契約）には労働基準法は適用されない

---

## 法定労働時間（労基法第32条）

| 区分 | 上限 |
|------|------|
| 1日 | 8時間 |
| 1週間 | 40時間（特例事業場は44時間） |

法定労働時間を超える場合は**36協定の締結・届出**が必要。

---

## 36協定の上限規制（2019年4月施行）

| 区分 | 上限 |
|------|------|
| 時間外労働（原則） | 月45時間・年360時間 |
| 時間外労働（特別条項あり） | 月100時間未満・年720時間以内 |
| 2〜6ヶ月の平均 | 月80時間以内 |
| 月45時間超えの月数 | 年6ヶ月以内 |

### 本システムへの適用（09-attendance.md）
- 月次残業時間集計・36協定上限との比較表示
- 月45時間・月80時間・月100時間の閾値アラート
- 年間残業時間の累計管理（360時間・720時間アラート）
- 特別条項適用月数のカウント（年6ヶ月超え警告）

---

## 割増賃金（労基法第37条）

| 種別 | 割増率 | 適用条件 |
|------|--------|---------|
| 時間外労働 | 25%以上 | 法定労働時間超え |
| 深夜労働 | 25%以上 | 22:00〜5:00 |
| 休日労働 | 35%以上 | 法定休日（週1日） |
| 月60時間超時間外 | 50%以上 | 中小企業は2023年4月から適用 |
| 時間外 + 深夜 | 50%以上 | 重複適用 |

### 本システムへの適用（09-attendance.md）
- 勤務種別ごとの割増率を自動計算
- 深夜勤務の自動判定（22:00〜5:00の時間帯）

---

## 年次有給休暇（労基法第39条）

| 継続勤務年数 | 付与日数（週5日・所定労働日数週5日） |
|------------|----------------------------------|
| 6ヶ月 | 10日 |
| 1年6ヶ月 | 11日 |
| 2年6ヶ月 | 13日 |
| 3年6ヶ月 | 14日 |
| 4年6ヶ月 | 16日 |
| 5年6ヶ月 | 18日 |
| 6年6ヶ月以上 | 20日 |

### 年5日取得義務（労基法第39条第7項）
年10日以上付与された社員に対し、**年5日の有給取得が使用者の義務**（2019年4月施行）。

### 本システムへの適用（09-attendance.md）
- 入社日・継続勤務年数から有給付与日数を自動計算
- 年5日未取得者への取得促進アラート（年度末2ヶ月前）
- 有給残日数・取得日数のリアルタイム表示

---

## 勤怠記録の保存・客観性

| 要件 | 内容 | 根拠 |
|------|------|------|
| 保存期間 | 5年（当面3年） | 労基法第109条 |
| 客観的記録 | タイムカード・PCログ等客観的方法による記録 | 厚労省「労働時間の適正な把握のために使用者が講ずべき措置に関するガイドライン」|
| 修正履歴 | 手入力修正時は修正前後・修正者・修正日時を記録 | 同ガイドライン |

### 本システムへの適用（09-attendance.md）
- 打刻データは変更不可（修正は別レコードで管理）
- 手動修正時は修正理由・修正者を必須記録
- 勤怠データは5年間保持（ソフトデリートのみ）

---

## SES特有の考慮事項

- **客先での勤怠管理**: 客先のタイムシートシステムとの二重管理が発生するケースあり。CSV取込機能でデータ連携
- **在宅・客先常駐の混在**: 同月内に在宅勤務と客先常駐が混在する場合の勤務場所記録
- **みなし労働時間**: 事業場外みなし労働時間制（客先常駐で適用されるケース）の対応
