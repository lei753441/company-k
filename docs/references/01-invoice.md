# インボイス制度（適格請求書等保存方式）

## 概要

| 項目 | 内容 |
|------|------|
| 正式名称 | 適格請求書等保存方式 |
| 根拠法令 | 消費税法 |
| 施行日 | 2023年10月1日 |
| 所管 | 国税庁 |

## 制度の概要

課税事業者が仕入税額控除を受けるために、**適格請求書発行事業者（インボイス発行事業者）**が発行した適格請求書（インボイス）の保存が必要となる制度。

---

## 適格請求書の必須記載事項（7項目）

1. 適格請求書発行事業者の氏名または名称および**登録番号**（T + 13桁）
2. 取引年月日
3. 取引内容（軽減税率対象品目の場合はその旨）
4. 税率ごとに区分した税抜価額または税込価額の合計額
5. 適用税率（10% または 8%）
6. **税率ごとに区分した消費税額等**
7. 書類の交付を受ける事業者の氏名または名称

---

## 免税事業者への対応（経過措置）

協力会社・個人事業主にインボイス未登録（免税事業者）が含まれる場合、仕入税額控除が制限される。

| 期間 | 控除可能割合 |
|------|------------|
| 2023年10月1日 〜 2026年9月30日 | 80% |
| 2026年10月1日 〜 2029年9月30日 | 50% |
| 2029年10月1日以降 | 0%（控除不可） |

---

## 本システムへの適用要件

### 協力会社・個人事業主管理（05-partner.md）
- インボイス登録ステータスを `registered / pending / not_registered` の3値で管理
- 登録番号形式：T + 13桁（法人番号と同一）
- 未登録の場合、支払処理画面に警告表示（仕入税額控除制限の旨）
- 国税庁公表サイトとの突合確認（Phase 1: 手動、Phase 2: API自動連携）

### 請求・支払管理（10-billing.md）
- 自社発行の請求書に7項目すべてを記載したPDF出力
- 協力会社からの受領請求書のインボイス番号確認・保存
- 免税事業者への支払時は経過措置に基づく控除割合を自動計算

### 経費管理（14-expense.md）
- 経費申請時に消費税率（10% / 8%軽減税率）を選択
- 協力会社経費のインボイス登録番号の有無に応じて仕入税額控除を判定

### 消費税の端数処理
- 請求書単位で切り捨て（消費税法の規定に準拠）
